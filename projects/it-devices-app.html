IT Devices Request App

The IT Devices Request App is a robust Power Apps solution designed to revolutionize the way IT departments manage device requests, approvals, and inventory. This application streamlines the entire lifecycle, from initial user request to final device allocation, significantly reducing manual overhead and improving overall efficiency.

Built on Power Apps Canvas for a flexible user interface, the application effectively utilizes SharePoint lists for data management. Automated workflows via Power Automate handle approval processes, notifications, and updates, ensuring timely delivery and transparent communication.

Admin Dashboard & Request Tracking

The main dashboard of the IT Devices Request App greets the administrator with a personalized message and offers an intuitive overview of their active requests. The "My Orders" section allows for a quick check of each device request's status, from pending approval to those already accepted by a manager or IT. Adjacent to this, the "Recently added devices" section displays new additions to the available equipment, enabling administrators to quickly submit new requests or monitor overall inventory.

  • Centralized request oversight for administrators
  • Real-time status tracking and device availability
  • Streamlined management of new device additions and user requests

Intuitive User Dashboard & Request Creation

The user-friendly dashboard provides a quick overview of pending requests and available devices. Users can easily initiate new device requests through a guided form, specifying device type, quantity, and purpose. This centralized system eliminates scattered email requests and ensures all necessary information is captured upfront.

  • Centralized request entry
  • Real-time status tracking
  • Simplified user experience

Request Management and Data Filtering

The "Orders" section provides a comprehensive dashboard for managing all IT device requests. Users, including administrators, can easily search, filter, and sort requests by various criteria such as device category, priority, status, or order date. This detailed view enables efficient tracking of each request's lifecycle, from submission to its final status, ensuring full transparency and control over the device distribution process.

  • Precise filtering and searching of requests
  • Clear status tracking for every order
  • Optimized IT inventory management

Approval Management and Request Authorization

The "Approvals" section serves as a dedicated panel for individuals responsible for authorizing device requests. It provides a clear overview of all pending requests requiring action, with filtering capabilities by category, priority, or status. A key feature is the two-step approval process: first, the request goes to the user's direct manager who submitted it, and only after their approval is it forwarded to the IT department for their acceptance. This module significantly accelerates the decision-making process, enabling quick verification and approval or rejection of requests, which guarantees a smooth workflow within the IT department.

  • Centralized dashboard for approval management
  • Streamlined, two-step request authorization process
  • Increased efficiency in decision-making

Equipment Management for IT Department

The "Equipment" section serves as a comprehensive tool for the IT department to efficiently manage the available device inventory. In this panel, administrators can easily view, add, edit, and remove devices from the list of available equipment. Thanks to intuitive filters and a search function, it is possible to quickly locate specific items, which significantly streamlines the process of updating the device database and maintaining its consistency with the actual stock level.

  • Centralized IT inventory management
  • Easy addition, editing, and removal of devices
  • Quick search and filtering of equipment

Overview of Available Devices and Request Submission

The "Equipment" section presents users with the available IT device inventory in a clear and visually appealing card format. Users can easily browse devices, filter them by category, and search for specific items. Each device card includes key information such as name, category, and price, and allows for direct submission of a request for that particular equipment. This intuitive view streamlines the process of selecting and ordering devices for employees.

  • Visual overview of available equipment
  • Easy filtering and searching of devices
  • Direct request submission from the product card
